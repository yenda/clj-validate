# validate

A Leiningen plugin to validate Clojure project

- checks dependencies
- checks code quality

## Usage

Use this for user-level plugins:

Put `[validate "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your `:user`
profile.

    $ lein validate
